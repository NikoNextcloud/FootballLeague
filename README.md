# А Група

Мобилно уеб приложение за мачове, резултати, класиране и отбори от българския футболен елит. Основните данни се четат от локални JSON файлове, обновявани автоматично от GitHub Actions bot-а.

## Отваряне на компютъра

Стартирайте `start-local.bat`. Не е нужна инсталация на програми или библиотеки (изисква наличен Python в Windows).

## Качване в GitHub Pages

1. Създайте ново публично хранилище в GitHub.
2. Качете съдържанието на тази папка в основната папка на хранилището.
3. Отворете **Settings → Pages**.
4. В **Build and deployment** изберете **Deploy from a branch**, `main` и `/ (root)`.
5. След запазване GitHub ще покаже адреса на приложението.

Приложението няма build стъпка, backend, proxy server или runtime зависимости. Това го прави подходящо за директно публикуване в GitHub Pages.

Началната страница включва непрекъсната лента с емблемите. Всеки клуб има отделен профил със стадион, статистика, описание и мачове. Стрелките в класирането сравняват позицията с предходния изигран кръг.

## Автоматично обновяване

GitHub Actions изпълнява `scripts/bulgarian_football_bot.py` на всеки 5 минути и обновява `data/bulgarian-football.json` от `bulgarian-football.com`. Това е основният локален източник за класиране, мачове, резултати, днешни мачове и голмайстори.

## Известия

Бутонът с камбанка включва известия за начало на мач, докато приложението работи. Истински push известия при затворено приложение и моментални известия за гол изискват отделен push сървър.

## Българските отбори в Европа

Табът „Европа“ съдържа българските участници и мачовете от локалния JSON, генериран от bot-а. Няма live scraping от браузъра и няма външен блок, който да показва празни данни.

## AiScore live панел

Когато започне мач от българската Първа лига, началният банер автоматично се заменя с вградената AiScore страница за Bulgarian First League. Панелът остава активен от пет минути преди началото до приблизително 165 минути след началния час. За проверка без изчакване може локално да се отвори `?live-preview=1`.
